India IPO Analytics Dashboard (2020–2025)

Interactive Streamlit dashboard analyzing 382 NSE mainboard IPOs from 2020–2025 using Python, Pandas, and Plotly.

Key Insights
IPO listings increased 575% from 2020 to 2025
Higher IPO volume correlated with lower average listing gains
July delivered the strongest average IPO returns
Paytm recorded one of the worst major IPO listings (-27.4%)


Features
IPO count analysis by year
Listing gain trends
Monthly seasonality insights
Top and worst-performing IPOs
Interactive Plotly visualizations
